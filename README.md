# Registration Form

A modern **Registration Form UI** built using **HTML, Tailwind CSS, and JavaScript**.
This project provides a clean user interface for creating a new account with basic validation and interactive features.

---

## Features

* Modern and responsive UI
* Built with **Tailwind CSS**
* Input fields for:

  * Name
  * Email
  * Password
* Password visibility toggle
* Basic form validation
* Clean card layout design
* Mobile-friendly interface

---

## Technologies Used

* **HTML5** – Structure of the webpage
* **Tailwind CSS** – Styling and responsive design
* **JavaScript** – Form validation and interaction
